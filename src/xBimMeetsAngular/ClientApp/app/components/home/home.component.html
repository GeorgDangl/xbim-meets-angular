xBim meets Angular!

Welcome to this small demo site. It's designed to show the usage of the xBim WebUI framework in comination with Angular. The project is available at Github.

It's been built with the following technologies to show how to integrate them:

  • xBimWebUI for client side BIM model visualization
  • ASP.NET Core and C# for cross-platform server-side code
  • Angular 2 and TypeScript for client-side code
  • Webpack for building and bundling client-side resources
  • Bootstrap for layout and styling
  • The Asp.Net Core Angular 2 Single Page Application Template served as base for this website
